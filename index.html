Aplikasi Tidak Dapat Dijalankan Langsung Dari File

Karena alasan keamanan, browser modern memblokir fitur seperti akses API dan penyimpanan lokal saat Anda membuka file HTML secara langsung. Ini bukan kesalahan pada aplikasi, tetapi tindakan keamanan standar.

Cara Memperbaikinya (Mudah!)

Anda perlu menyajikan file ini dari server web lokal sederhana. Berikut adalah dua cara umum:

Opsi 1: Menggunakan Python

Jika Anda memiliki Python, buka terminal atau command prompt di direktori file ini dan jalankan:

python -m http.server 8000

Kemudian, buka browser Anda ke: http://localhost:8000

Opsi 2: Menggunakan Node.js

Jika Anda memiliki Node.js, buka terminal di direktori file ini dan jalankan:

npx serve

Terminal akan memberi Anda alamat lokal untuk dibuka, biasanya http://localhost:3000

Setelah Anda menjalankan aplikasi dari alamat http://localhost, pesan ini akan hilang dan aplikasi akan berfungsi dengan benar.
